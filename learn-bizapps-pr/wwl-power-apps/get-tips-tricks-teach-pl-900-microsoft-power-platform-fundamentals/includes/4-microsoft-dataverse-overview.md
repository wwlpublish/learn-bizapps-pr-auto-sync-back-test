### Module 2 Tips and Tricks 

- Module 2 lends itself well to white boarding and organizational demos 

- Clarify the importance of understanding the vocabulary around the Common Data Service 

 - Consider creating a mini-review game to ensure students are retaining their knowledge of the different terms 

- Describe the difference between environments, tables, columns, and relationships 

- Highlight differences between Microsoft Dataverse and the Common Data Model 

 - Consider using [Microsoft GitHub Common Data Model](https://aka.ms/githubCDM) as an example 

- Emphasize that the Common Data Service is the foundation for Microsoft Power Platform
