As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn the business value and features of Power BI. Students will also explore how Power BI works and looks from the user’s perspective. 

This module consists of two lessons: 

**Lesson 1** – Power BI overview 

**Lesson 2** – How to Build a Simple Dashboard

### Module 5 Tips and Tricks 

- Consider discussing concepts from the DP-900 Microsoft Azure Data Fundamentals course for students to understand why visualizations are needed 

- Explore the different data connectors and discuss how they can be used in student life 

- Use the same set of data and use different templates to show the capacity of the system and choosing the best visualization 

- When building the dashboard, be sure to encourage students to create a viable board versus adding different graphics 

- Consider running a general characteristics survey on students. For example, hobbies, hometown, height, etc. Next, as part of the demonstration builds a Power BI dashboard using visualization options
