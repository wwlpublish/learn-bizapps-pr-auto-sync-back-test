One advantage of Power Virtual Agents is the simplicity that it offers you when you're using other technologies to provide a more personalized and tailored experience. Services such as Azure Cognitive Services for Language lets you bring in features, such as knowledge bases, and extract personal data to create a richer user experience.

In this module, you learned how to use Azure Cognitive Services for Language in a Power Virtual Agent bot, including:

- Discovering the reasons why using other services is important to your bot functionality.

- Learning about Azure Cognitive Services for Language and its capabilities.

- Reviewing how to create a bot and prepare it for using Azure Cognitive Services for Language.

- Examining how to call your Azure Cognitive Services for Language service with a Power Automate flow, from within a bot.
