One of the many advantages of Power Virtual Agents is the simplicity it provides when using other technologies to provide a more personalized and tailored experience. Services such as Azure Cognitive Service for Language lets you bring in things like knowledge bases and extract Personal Identifying Information to create a richer user experience.

Over the course of this module, we examined how to use Azure Cognitive Service for Language in a Power Virtual Agent bot, including:

- Exploring why using other services is important to your bot functionality.

- Examining what Azure Cognitive Service for language is and what some of the capabilities available are.

- Reviewing how to create a bot and get it ready to use Azure Cognitive Service for Language.

- Examining how to call your Azure Cognitive Service for Language service with a Power Automate flow, from within a bot.
