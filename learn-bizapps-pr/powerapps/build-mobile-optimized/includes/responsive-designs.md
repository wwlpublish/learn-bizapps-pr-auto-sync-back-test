The following video demonstrates the process of creating a blank canvas app by using elements of a responsive design. This process provides an optimized experience for a person who uses a tablet or phone with different layouts and dimensions.

This demonstration uses a tablet design instead of a mobile design to provide more space on a single screen so that the designer can view all elements on a single screen for demonstration purposes.
