In a model-driven app, data is stored in the tables within Dataverse, and forms can provide the user interface (UI) to interact with the data. Controls on the form allow users to visualize data within table columns. You can also interact with the data by using the controls.

Apart from the regular controls, like tab, section, and input controls, model-driven forms include some special controls to manage relational data and cater to typical business requirements. We'll discuss these special controls in this course.
