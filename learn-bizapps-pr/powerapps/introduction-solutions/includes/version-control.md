## Power Platform CLI

Microsoft Power Platform CLI is a simple, one-stop developer CLI that empowers developers and ISVs to perform various operations in Microsoft Power Platform related to environment lifecycle, authentication, and work with Microsoft Dataverse environments, solution packages, portals, code components, and additional collection of ALM-related tasks for Power Platform.

It's compatible with Linux, Mac, and Windows and can be installed through automation, making it ideal for DevOps processes.
