In this module you learned that designing the app is an important part of the process, where you start to specify what you'll be creating in the making phase based on the information you've gathered in the planning phase.

Now that you've reviewed this module, you should be able to:

-   Analyze the different tasks to be performed by the users using the app

-   Prepare the specifications and designs of your app

-   Decide what the user experience will be like
