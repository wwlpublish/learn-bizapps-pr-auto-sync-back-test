The Microsoft Cloud for Retail solution is composed of various integrated Microsoft technologies (Dynamics 365, Microsoft 365, and Azure) that are tailored specifically for retail organizations.

Microsoft Cloud for Retail has specific product requirements to light up each scenario and feature. These Dynamics, Azure, and Microsoft 365 pre-requisites are outlined in the tables below, some of which are optional and won't be included in this learning path.
